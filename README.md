## 🎯 Design Philosophy

### Typography
- **Font**: Inter (300-900 weights)
- **Hierarchy**: Ultra-bold headings (900), medium body (400-500)
- **Spacing**: Tight tracking on large text (-0.04em), relaxed on body
- **Rendering**: Antialiased with optimized font features

### Colors
- **Background**: Pure black (#000000)
- **Accent**: Deep indigo (rgb(79, 70, 229))
- **Glow**: Dark purple (rgb(67, 56, 202))
- **Text**: White with opacity variations

### Animation Principles
- **Smooth**: Spring physics for natural movement
- **Subtle**: Reduced motion for accessibility
- **Purposeful**: Animations enhance, not distract
- **Performance**: GPU-accelerated transforms

**Want to modify or contribute?**

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/adityayadav161003/ay-systems.git

# Navigate to project
cd ay-systems

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build for Production

```bash
# Create optimized production build
npm run build

# Start production server
npm start
```
